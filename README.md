# Angular-Apps smarter machen mit Gen AI: Lokal und offlinefähig - Hands-on Workshop!

[BASTA! Spring 2026](https://basta.net/frankfurt/), 2. März 2026, 09:00–17:00

Trainer: Christian Liebel ([@christianliebel](https://x.com/christianliebel))

---

Generative AI ist längst im Alltag angekommen – und jetzt auch in Angular! In diesem Workshop lernen Sie, wie Sie mit Transformers.js und der Prompt API KI-Funktionalität direkt in Ihre Anwendungen integrieren. Entwickeln Sie einen Chatbot und Formulare, die sich dank AI selbst ausfüllen. Lokal, offlinefähig und hands-on: So bringen Sie Generative AI praxisnah in Ihr Frontend.

## Installationsanweisungen

Wenn Sie mitentwickeln möchten, bringen Sie bitte ein leistungsfähiges Gerät mit Windows oder macOS, eine aktuelle Version von Chrome, Node.js, einen Editor Ihrer Wahl und mindestens 20 GB freien Festplattenspeicher mit.

Idealerweise ist dies Ihr privates Gerät, da Gruppenrichtlinien oder Unternehmensproxys oft Probleme verursachen.

Nachdem Sie das Repository geklont haben, führen Sie bitte die folgenden Befehle in der Kommandozeile aus:

```sh
cd basta-spring-2026-genai
npm i
```
